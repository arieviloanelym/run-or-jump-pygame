# 🏃‍♂️ Run or Jump

Jogo de sobrevivência arcade onde o objetivo é desviar de bolas de fogo o maior tempo possível.

## 🛠️ Tecnologias
* Python 3.x
* Pygame-ce

## 🎮 Como Jogar
1. Certifique-se de ter o Python instalado.
2. Instale o Pygame: `pip install pygame-ce`
3. Execute o jogo: `python jogo.py`

**Comandos:**
* `ESPAÇO`: Pular / Iniciar Jogo
* `M`: Mutar/Desmutar música

## 📋 Requisitos do Projeto
- [x] Controle do jogador
- [x] Desafio progressivo
- [x] Condição de vitória (Demo 30s)
- [x] Condição de derrota (Colisão)
